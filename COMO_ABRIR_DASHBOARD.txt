🔧 SI HAY PROBLEMAS:
───────────────────────────────────────────────────────────────

PROBLEMA 1: "No se puede conectar" o "This site can't be reached"
─────────────────────────────────────────────────────────────────
SOLUCIÓN:
  1. Verifica que la ventana de PowerShell con Streamlit está abierta
  2. En esa ventana busca el texto "Local URL: http://localhost:8501"
  3. Si no aparece, cierra la ventana y ejecuta:

     cd "C:\Users\JHON LEON\Documents\Proyectos\Predictor de marcadores\sports-forecasting-pro"
     python -m streamlit run app.py


PROBLEMA 2: "No encuentro reports/backtest_log.csv"
─────────────────────────────────────────────────────────────────
SOLUCIÓN:
  El dashboard necesita datos del backtest. Ejecuta primero:

     python scripts/backtest_all_markets.py

  Luego vuelve a abrir el dashboard.


PROBLEMA 3: "Port 8501 is already in use"
─────────────────────────────────────────────────────────────────
SOLUCIÓN:
  Ya hay un Streamlit corriendo. Usa otro puerto:

     python -m streamlit run app.py --server.port 8502

  Luego abre: http://localhost:8502


PROBLEMA 4: La página carga pero no muestra datos
─────────────────────────────────────────────────────────────────
SOLUCIÓN:
  1. Presiona "R" en el teclado (recarga el dashboard)
  2. O refresca la página (F5)
  3. Si no funciona, verifica que existe:
     - data/processed/matches.parquet
     - reports/backtest_log.csv
